static

Permet d'écrire une clé dans un fichier INI,. 

- si seul la section est complèté elle sera supprimée ($cle et $ValeurAEcrire = NULL).
- si la section et la clé sont remplis mais pas la valeur , la clé sera supprimé ($ValeurAEcrire = NULL).
- si tous les paramètre sont précisé alors la clé sera ajouter ou modifié si elle est déjà présente dans le fichier.
Parameters
$SectionChaine contenant le nom de la section à écrire
$CleChaine contenant le nom de la clé à écrire
$ValeurAEcrireChaine la valeur à dans le fichier
$FichierChaine contenant le nom et le chemin du fichier INI
Returns
Boolean True si l'écriture à été faite sinon False
